Easy Popover Getting Started and Demos

EasyPopover - is easy and simple library for popover.
It dosn't depend on a specific library.

Purpose of this library

It is intended to be used to describe elements on the pages of your site. On load page, popup automatically.

If you close the popover, it Never displayed.

If you close, save the flag you closed to local storage.

Demos

Display popover at specific ellemnt.

Getting statrted

1. Download and install

First of all we need to download required files.

  • We can download them from GitHub repository
  • Or we can install them via npm in terminal.

And use files in the dist/ folder.

2. Include Files To Website

After that we need to include JS files to our website. In your html file. 

<!DOCTYPE html>
<html>
<head>
  ...
  <link rel="stylesheet" href="path/to/popover.css">
</head>
<body>
  ...
  <script src="path/to/EasyPopover.js"></script>
  <script>
    document.addEventListener('DOMContentLoaded', function() {
      EasyPopover.init()
    })
  </script>
</body>
</html>
